Object Tracking Robot

Object Tracking Robot (Using Image Processing)

In this project, the variation in horizontal and vertical axis of tracked object generate control signal which is send to the controller wirelessly. The captured images of the object are processed using software MATLAB. Depending on the change in position of object, proper commands are given to the robot to track the object. The motion of the robot is controlled by a microcontroller based on the control signals received directly.

AAAI 2018

"Consonant-Vowel Sequences as Subword Units for Code-Mixed Languages"

In this research work, we developed a state-of-art model for identifying sentiment in Hindi-English code-mixed language. We introduced new phonemic sub-word units for Hindi-English code-mixed text along with a hierarchical deep learning model which uses these sub-word units for predicting sentiment.

ICON 2017

"A pessimist sees the difficulty in every opportunity; an optimist sees the opportunity in every difficulty" -- Understanding the psycho-sociological influences to it

This paper presents an empirical study to understand how psycho-sociological factors influence on optimism/pessimism at the individual level.
